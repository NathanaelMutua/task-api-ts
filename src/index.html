My Task API with Typescript

Description

This project is a simple RESTful API built with TypeScript that allows you to create, read, update, and delete tasks. It demonstrates core CRUD operations.

  1. Copy the link to this site
  2. Visit Postman or using curl to implement the endpoints
  3. Use the endpoint references as described below

API Implementation

  • POST /tasks
  • Creating a task http://127.0.0.1:7000/tasks
  • GET /tasks
  • Getting all tasks http://127.0.0.1:7000/tasks
  • GET /tasks/:id
  • Getting a specific task http://127.0.0.1:7000/tasks/""
  • PATCH /tasks/:id
  • Updating a task http://127.0.0.1:7000/tasks/""
  • DELETE /tasks/:id
  • Deleting a task http://127.0.0.1:7000/tasks/""

Technologies

  • Prisma
  • Typescript
  • PostgreSQL
  • Node.JS
  • Express.JS
